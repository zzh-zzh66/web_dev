# 用户手册

## 1. 系统简介

### 1.1 产品概述

家族族谱系统是一款帮助家族记录、管理和展示家族谱系信息的数字化平台。通过本系统，您可以：

- 记录家族成员的基本信息和生平事迹
- 建立和维护家族成员之间的关系
- 以可视化方式展示家族族谱
- 保存和分享家族老照片
- 永久保存家族历史，传承家族文化

### 1.2 系统要求

| 项目 | 要求 |
|------|------|
| 推荐浏览器 | Chrome 90+、Firefox 88+、Safari 14+、Edge 90+ |
| 网络要求 | 稳定的互联网连接 |
| 分辨率 | 最佳显示 1920x1080 及以上 |

## 2. 快速入门

### 2.1 账号注册

1. 打开系统登录页面
2. 点击"注册账号"链接
3. 填写手机号、密码、验证码
4. 阅读并同意《用户协议》
5. 点击"注册"完成

### 2.2 首次登录

1. 输入用户名和密码
2. 点击"登录"按钮
3. 首次登录后，建议完善个人资料

### 2.3 创建/加入家族

```
新用户首次登录后：
1. 选择"创建新家族"或"加入已有家族"
2. 创建家族：填写家族名称、起源地等信息
3. 加入家族：输入家族邀请码或由管理员邀请
```

## 3. 功能使用指南

### 3.1 成员管理

#### 添加成员

```
步骤：
1. 进入"成员管理"页面
2. 点击"添加成员"按钮
3. 填写成员信息：
   - 姓名（必填）
   - 性别（必填）
   - 出生日期
   - 父亲姓名（选择或新建）
   - 母亲姓名（选择或新建）
   - 配偶姓名
   - 职业、出生地等
4. 点击"保存"
```

**注意事项：**
- 带*号的为必填项
- 父亲/母亲信息可先不填，后续可在关系管理中添加
- 出生日期格式：YYYY-MM-DD

#### 编辑成员

```
步骤：
1. 在成员列表中找到要编辑的成员
2. 点击成员卡片上的"编辑"按钮
3. 修改相关信息
4. 点击"保存"
```

#### 删除成员

```
步骤：
1. 在成员列表中找到要删除的成员
2. 点击"删除"按钮
3. 系统提示确认删除操作
4. 确认后删除

注意：
- 如果成员有子女，系统将阻止删除
- 删除操作不可恢复，请谨慎操作
```

### 3.2 关系管理

#### 建立关系

```
父子关系：
1. 进入"关系管理"页面
2. 点击"添加关系"按钮
3. 选择成员A和成员B
4. 选择关系类型（父子/母子）
5. 点击"保存"

配偶关系：
1. 进入成员详情页
2. 点击"添加配偶"
3. 选择或创建配偶信息
4. 保存
```

#### 关系类型说明

| 关系类型 | 说明 |
|----------|------|
| 父子 | 父亲与子女的关系 |
| 母子 | 母亲与子女的关系 |
| 配偶 | 夫妻关系 |
| 兄弟姐妹 | 同父母的兄弟姊妹 |

#### 关系规则

```
1. 循环关系禁止：不能创建A是B的父亲，B是A的父亲这样的循环
2. 年龄逻辑：父母的出生日期必须早于子女
3. 近亲限制：直系血亲之间不能建立配偶关系
```

### 3.3 族谱展示

#### 查看族谱

```
步骤：
1. 进入"族谱展示"页面
2. 系统默认显示完整族谱树
3. 点击节点可查看成员详情
4. 使用鼠标滚轮缩放
5. 拖拽移动视图
```

#### 视图切换

```
支持以下视图：
- 竖向树形图：传统族谱样式，自上而下
- 横向树形图：自左向右展开
- 时间轴视图：按出生时间顺序展示
- 表格视图：以表格形式展示成员列表
```

#### 导出族谱

```
步骤：
1. 进入族谱展示页面
2. 点击右上角"导出"按钮
3. 选择导出格式（PDF/图片）
4. 配置导出选项：
   - 起始成员
   - 展示深度
   - 是否包含照片
5. 点击"导出"
6. 系统生成文件后自动下载
```

### 3.4 照片管理

#### 上传照片

```
步骤：
1. 进入成员详情页或"照片管理"页面
2. 点击"上传照片"按钮
3. 选择本地图片文件
4. 填写照片描述（可选）
5. 点击"上传"

支持格式：JPG、PNG
文件大小：单张不超过10MB
```

#### 照片预览和下载

```
预览：
1. 在照片列表中点击照片
2. 进入全屏预览模式
3. 可放大、缩小查看

下载：
1. 在照片预览页面
2. 点击"下载"按钮
3. 照片保存到本地
```

### 3.5 数据导入导出

#### Excel导入

```
步骤：
1. 进入"数据管理"页面
2. 点击"Excel导入"
3. 下载导入模板
4. 按模板格式填写成员信息
5. 上传填写好的Excel文件
6. 系统预览导入数据
7. 确认无误后点击"确认导入"
```

**Excel模板说明：**

| 列名 | 说明 | 必填 |
|------|------|------|
| 姓名 | 成员姓名 | 是 |
| 性别 | 男/女 | 是 |
| 出生日期 | YYYY-MM-DD格式 | 否 |
| 父亲姓名 | 如有关联 | 否 |
| 配偶姓名 | 如有关联 | 否 |
| 辈分 | 数字 | 否 |
| 出生地 | 籍贯 | 否 |

#### 数据导出

```
步骤：
1. 进入"数据管理"页面
2. 点击"Excel导出"
3. 选择导出范围（全部/筛选）
4. 点击"导出"
5. 系统生成Excel文件下载
```

### 3.6 搜索功能

#### 快速搜索

```
使用方法：
1. 在顶部搜索框输入关键词
2. 系统实时显示搜索结果
3. 点击结果跳转到成员详情

搜索范围：姓名、配偶姓名、出生地、职业等
```

```
步骤：
1. 点击搜索框旁的"高级搜索"
2. 在搜索面板中填写条件：
   - 姓名
   - 性别
   - 出生年份范围
   - 辈分
   - 出生地
   - 职业
3. 点击"搜索"
4. 查看筛选结果
```

## 5. 常见问题

### 5.1 登录问题

**Q: 忘记密码怎么办？**
```
1. 在登录页面点击"忘记密码"
2. 输入注册手机号
3. 接收验证码
4. 设置新密码
```

**Q: 账号被锁定怎么办？**
```
连续5次登录失败后，账号将被锁定30分钟。
等待锁定时间结束后重试，或联系管理员解锁。
```

### 5.2 数据问题

**Q: 如何添加不在世的家族成员？**
```
在添加成员时，将状态选择为"去世"，
然后填写生卒日期即可。
```

**Q: 如何修改成员的头像？**
```
1. 进入成员详情页
2. 点击头像区域
3. 选择新图片上传
4. 保存修改
```

**Q: 导入数据时报错怎么办？**
```
常见错误：
1. 姓名必填 - 检查是否有空姓名
2. 性别格式错误 - 只允许"男"或"女"
3. 日期格式错误 - 使用YYYY-MM-DD格式
4. 重复成员 - 检查是否有重复姓名

请修正后重新导入。
```

### 5.3 族谱展示问题

**Q: 族谱显示不完整？**
```
可能原因：
1. 部分成员关系未建立
2. 辈分信息缺失
3. 数据量过大，加载超时

解决方法：
1. 在关系管理中完善成员关系
2. 为成员填写辈分信息
3. 尝试缩小展示范围
```

**Q: 如何找到族谱的起始点？**
```
1. 在族谱展示页面
2. 点击"定位祖先"按钮
3. 系统自动定位到最早的祖先节点
```
