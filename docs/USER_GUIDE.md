# 用户手册

## 1. 系统简介

### 1.1 产品概述

家族族谱系统是一款帮助家族记录、管理和展示家族谱系信息的数字化平台。通过本系统，您可以：

- 记录家族成员的基本信息
- 以可视化方式展示家族族谱
- 管理家族成员关系

### 1.2 系统要求

| 项目 | 要求 |
|------|------|
| 推荐浏览器 | Chrome 90+、Firefox 88+、Safari 14+、Edge 90+ |
| 网络要求 | 稳定的互联网连接 |
| 分辨率 | 最佳显示 1920x1080 及以上 |

## 3. 功能使用指南

### 3.1 成员管理

#### 添加成员

```
步骤：
1. 进入"成员管理"页面
2. 点击"添加成员"按钮
3. 填写成员信息：
   - 姓名（必填）
   - 性别（必填）
   - 出生日期
   - 父亲姓名（选择或新建）
   - 母亲姓名（选择或新建）
   - 配偶姓名
   - 职业、出生地等
4. 点击"保存"
```

**注意事项：**
- 带*号的为必填项
- 父亲/母亲信息可先不填，后续可在关系管理中添加
- 出生日期格式：YYYY-MM-DD

#### 编辑成员

```
步骤：
1. 在成员列表中找到要编辑的成员
2. 点击成员卡片上的"编辑"按钮
3. 修改相关信息
4. 点击"保存"
```

#### 删除成员

```
步骤：
1. 在成员列表中找到要删除的成员
2. 点击"删除"按钮
3. 系统提示确认删除操作
4. 确认后删除

注意：
- 如果成员有子女，系统将阻止删除
- 删除操作不可恢复，请谨慎操作
```

### 3.2 族谱展示

#### 查看族谱

```
步骤：
1. 点击左侧导航"族谱展示"
2. 进入族谱展示页面
3. 查看家族树形图结构

操作说明：
- 鼠标滚轮：缩放族谱
- 鼠标拖拽：移动视图
- 双击画布：重置视图到初始位置
- 单击节点：查看成员详情
- 双击节点：展开/折叠子树
```

#### 筛选成员

```
步骤：
1. 在左侧筛选面板选择条件：
   - 辈分筛选（可多选）
   - 性别筛选（男/女/全部）
   - 状态筛选（在世/已故/全部）
2. 筛选条件实时生效
3. 点击"重置"清除所有筛选
```

#### 辈分导航

```
步骤：
1. 在左侧辈分导航栏查看各辈分统计
2. 点击某辈分卡片筛选显示该辈分成员
3. 再次点击取消筛选
```

## 4. 常见问题

### 4.1 登录问题

**Q: 忘记密码怎么办？**
```
1. 在登录页面点击"忘记密码"
2. 输入注册手机号
3. 接收验证码
4. 设置新密码
```

**Q: 账号被锁定怎么办？**
```
连续5次登录失败后，账号将被锁定30分钟。
等待锁定时间结束后重试，或联系管理员解锁。
```

### 4.2 数据问题

**Q: 如何添加不在世的家族成员？**
```
在添加成员时，将状态选择为"去世"，
然后填写生卒日期即可。
```

**Q: 如何修改成员的头像？**
```
1. 进入成员详情页
2. 点击头像区域
3. 选择新图片上传
4. 保存修改
```

### 4.3 族谱展示问题

**Q: 族谱显示不完整？**
```
可能原因：
1. 部分成员关系未建立
2. 辈分信息缺失
3. 数据量过大，加载超时

解决方法：
1. 在关系管理中完善成员关系
2. 为成员填写辈分信息
3. 尝试缩小展示范围
```

**Q: 如何找到族谱的起始点？**
```
1. 在族谱展示页面
2. 点击"定位祖先"按钮
3. 系统自动定位到最早的祖先节点
```
