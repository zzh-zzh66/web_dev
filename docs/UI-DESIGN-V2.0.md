## 一、设计理念

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| 家族传承感 | 融入中国传统家族文化元素，体现厚重感与归属感 |
| 简洁易用 | 考虑到长辈用户(50+岁)的使用习惯，界面简洁、操作直观 |
| 内容优先 | 突出展示家族故事、生平记录、照片等核心内容 |
| 情感连接 | 通过温暖的色彩、圆润的界面元素，营造亲密的家族氛围 |
| 一致性 | 与V1.0现有系统保持视觉和交互风格统一 |

### 1.2 中国传统元素融合

**配色：**
- 中国红 `#C41E3A` - 用于喜庆场景、点赞、重要标记
- 靛青蓝 `#2C5F8D` - 主色调延伸，沉稳大气
- 金色 `#D4A017` - 族长标识、荣誉标记
- 水墨灰 `#4A4A4A` - 正文颜色，如传统书法
- 宣纸白 `#F9F6F0` - 页面背景，仿宣纸质感

**图案：**
- 云纹装饰 - 用于分隔线、卡片装饰边角
- 回纹边框 - 用于重要卡片、族长主页
- 印章元素 - 用于认证标记、官方内容标识
- 卷轴式时间轴 - 生平记录展示

**字体：**
- 标题可使用书法体(如思源宋体)，增强文化底蕴
- 正文保持现代无衬线字体，确保可读性

### 1.3 大字体适配方案

针对族长等年长用户(50+岁)，提供大字体模式:

| 设置项 | 标准模式 | 大字体模式 |
|--------|---------|-----------|
| 基础字号 | 14px | 16px |
| H1页面标题 | 24px | 28px |
| H2模块标题 | 20px | 24px |
| 正文内容 | 14px | 16px |
| 辅助文字 | 12px | 14px |
| 按钮文字 | 14px | 16px |
| 行高 | 1.6 | 1.8 |
| 按钮高度 | 40px | 48px |
| 输入框高度 | 40px | 48px |

**交互说明：**
- 对话框宽度：PC端 720px，移动端全屏
- 支持拖拽上传图片
- 图片上传时显示进度条
- 视频上传时显示压缩进度
- @提醒：输入@后弹出家族成员列表
- 话题标签：输入#后显示热门话题建议
- 发布按钮：输入内容后变为可点击状态

### 3.3 点赞交互动画流程

**默认状态：**
```
+------+
|  赞  |  89     (灰色图标，灰色数字)
+------+
```

**用户点击[赞]按钮：**

| 帧 | 时间 | 动画效果 |
|----|------|---------|
| 第1帧 | 0ms | 图标变为实心 #409EFF，放大到 1.3x |
| 第2帧 | 100ms | 图标回弹到 1.1x，数字+1 |
| 第3帧 | 200ms | 图标回到正常大小 1.0x，显示完成状态 |

**最终状态：**
```
+------+
|  赞  |  90     (蓝色图标，蓝色数字)
+------+
```

**取消点赞：**
- 图标缩小到 0.9x
- 数字-1
- 图标恢复灰色
- 时长：200ms ease-in

**双击点赞（快捷操作）：**
- 在内容图片/视频上双击
- 出现放大的点赞图标动画
- 从双击位置飞出爱心粒子效果
- 自动为该条内容点赞

**粒子效果：**
- 5-8个小爱心从双击位置向四周飞散
- 颜色：#409EFF, #F56C6C, #E6A23C（随机）
- 大小：8px - 16px（随机）
- 动画：飞散后渐隐消失
- 时长：600ms ease-out

### 3.4 无限滚动加载流程

**触发条件：**
- 用户滚动到距离底部 500px 时触发
- 防抖处理：两次加载间隔至少 500ms

**加载状态展示：**

1. **加载中状态：**
```
+-----------------------------------+
|                                   |
|       [旋转loading图标]           |
|       正在加载更多内容...          |
|                                   |
+-----------------------------------+
```

2. **加载完成：**
- 新内容以淡入动画出现 (fade-in, 300ms)
- 平滑插入到列表底部

3. **没有更多数据：**
```
+-----------------------------------+
|                                   |
|       ------ 已经到底了 ------     |
|                                   |
+-----------------------------------+
```

4. **加载失败：**
```
+-----------------------------------+
|                                   |
|       加载失败，请重试            |
|           [点击重试]              |
|                                   |
+-----------------------------------+
```

**性能优化：**
- 使用 IntersectionObserver API 检测滚动位置
- 预加载：提前200px触发加载，减少等待感
- 请求取消：如果用户快速滚动，取消未完成的请求
- 虚拟列表：当内容超过50条时，使用虚拟滚动优化
