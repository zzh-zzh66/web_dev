## 一、内容分类体系

### 1.1 系统预置分类

系统提供6个预置动态分类，覆盖族谱社交场景的核心内容类型：

| 分类名称 | 编码 | 图标 | 颜色 | 说明 | 排序 |
|---------|------|------|------|------|------|
| 生平日志 | LIFE_LOG | 日常记录 | #4A90D9 | 记录生活中的点点滴滴 | 1 |
| 家族故事 | FAMILY_STORY | 家族历史 | #E74C3C | 分享家族的历史和故事 | 2 |
| 照片墙 | PHOTO_WALL | 照片集锦 | #2ECC71 | 照片集锦，美好瞬间 | 3 |
| 视频集 | VIDEO_COLLECTION | 视频分享 | #9B59B6 | 视频分享，精彩时刻 | 4 |
| 心情随笔 | MOOD_DIARY | 心情感悟 | #F39C12 | 心情记录，随笔感悟 | 5 |
| 重要事件 | IMPORTANT_EVENT | 人生节点 | #E67E22 | 人生重要节点记录 | 6 |

### 1.2 内容类型

| 类型编码 | 说明 | 适用场景 |
|---------|------|---------|
| TEXT | 纯文字 | 感悟、故事、日常记录 |
| IMAGE | 图片 | 照片墙、活动记录 |
| VIDEO | 视频 | 旅行Vlog、活动视频 |
| LIFE_EVENT | 人生事件 | 出生、结婚、升学等 |

### 1.3 可见范围

| 范围 | 说明 |
|------|------|
| PUBLIC | 所有人可见（含游客） |
| FAMILY | 同家族成员可见 |
| RELATIVE | 仅亲戚（直系+旁系）可见 |
| PRIVATE | 仅自己可见 |

## 三、内容推荐算法

### 3.1 推荐原则

V2.0内容系统采用**按时间顺序**的线性推荐策略，符合家族社交场景的需求。

### 3.2 推荐规则

```
推荐列表 = 按 create_time 降序排序

过滤条件：
1. 仅展示 status = 'PUBLISHED' 的内容
2. 可见性过滤：
   - PUBLIC: 所有用户可见
   - FAMILY: 仅同 family_id 用户可见
   - RELATIVE: 仅通过亲属关系验证的用户可见
3. 排除已屏蔽用户的内容
4. 置顶内容（is_top = 1）始终排在最前面
```

### 3.3 排序优先级

```
1. 置顶内容（is_top = 1）
2. 精华内容（is_essence = 1）
3. 按 create_time 降序排列
```

### 3.4 个性化增强（V2.1规划）

- 基于用户兴趣标签匹配推荐
- 基于互动历史（点赞、评论、收藏）推荐相似内容
- 家族活动相关内容优先展示

## 四、内容审核规则

### 4.1 审核触发条件

以下内容发布时触发审核流程：

| 触发条件 | 说明 |
|---------|------|
| 首次发布用户 | 新用户前3条内容需审核 |
| 关键词匹配 | 包含敏感词库中的关键词 |
| 举报触发 | 被举报次数 >= 3 次自动进入审核 |
| 批量发布 | 短时间内发布超过5条内容 |

### 4.2 审核规则

```
内容审核规则：
1. 敏感词过滤：政治敏感、暴力、色情、赌博等
2. 家族规范：不尊重长辈、破坏家族和谐的内容
3. 广告过滤：商业推广、营销内容
4. 隐私保护：未经同意发布他人隐私信息

审核结果处理：
- 通过：status = 'PUBLISHED'
- 拒绝：status = 'REJECTED'，通知发布者原因
- 修改建议：status = 'DRAFT'，建议修改后重新发布
```

### 4.3 举报处理流程

```
用户举报 -> 举报数 >= 3 -> 自动隐藏（status = 'HIDDEN'）
    -> 族长/管理员审核 -> 确认违规 -> 永久删除
                        -> 误报 -> 恢复显示
```

## 五、内容增长策略

### 5.1 用户引导策略

**新用户首次访问**：
1. 展示示范用户（张德厚、张建国、张晓明）的主页内容
2. 提供内容发布模板和引导提示
3. 推荐关注的热门话题

**新用户首次发布**：
1. 提供6种内容分类选择引导
2. 提供内容标题和正文的写作建议
3. 推荐添加1-3张照片增强内容丰富度

### 5.2 话题运营策略

| 话题 | 运营方式 | 目标 |
|------|---------|------|
| #家族记忆 | 族长发起话题，分享家族历史 | 促进家族文化传承 |

- **互动激励**：评论、点赞、收藏等互动行为获得积分
- **创作激励**：发布原创内容获得积分
- **精华内容**：被设为精华的内容创作者获得额外奖励
- **月度榜单**：评选"最佳家族记录者"、"最热心评论员"等

### 5.4 内容沉淀策略

- 定期整理优质内容形成《家族年鉴》
- 老照片数字化保存，建立家族数字档案馆
- 重要家族事件（祭祖、聚会）内容自动归档
